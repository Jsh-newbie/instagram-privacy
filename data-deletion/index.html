최종 수정일: 2026년 4월 13일

수집하는 데이터

본 서비스("고교학점제 탐구 가이드")는 Instagram API를 통해 다음 데이터에 접근합니다:

  • Instagram 사용자 이름 및 사용자 ID
  • 게시물 댓글 내용
  • 다이렉트 메시지(DM) 내용

위 데이터는 자동 응답 처리 후 서버에 장기 저장하지 않습니다.

데이터 삭제 요청 방법

아래 방법 중 하나로 데이터 삭제를 요청할 수 있습니다:

  1. Instagram DM — 본 서비스의 Instagram 계정으로 "데이터 삭제 요청" 메시지를 보내주세요.
  2. 이메일 — admin@thejiniuslab.com으로 삭제 요청 이메일을 보내주세요.

삭제 처리 절차

  1. 삭제 요청 접수 확인 (1영업일 이내)
  2. 요청자 본인 확인
  3. 관련 데이터 삭제 처리 (요청일로부터 7일 이내)
  4. 삭제 완료 통보

참고 사항

  • 삭제 요청 시 Instagram 사용자 이름을 함께 알려주시면 빠르게 처리됩니다.
  • 이미 자동 삭제된 데이터는 별도 삭제 처리가 불필요할 수 있으며, 이 경우 안내드립니다.
  • 법령에 따라 보관이 필요한 데이터는 해당 기간 경과 후 삭제됩니다.
